# Checkbox

Aura UI Checkbox component.

## Installation

```sh
npm install @aura-ui/checkbox
# or
yarn add @aura-ui/checkbox
```

## Import Components

```jsx
import {
Checkbox
} from '@aura-ui/checkbox'
```

## Component

`Checkbox`: A control that allows users to toggle between checked and unchecked states..
